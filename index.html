Data Management

Data needs to be collected, stored in an adequate format, and properly cleansed before the analysis stage

  • Gather data from different sources including APIs, databases, and spreadsheets.
  • Collect data through experiments and surveys designed with scientific rigor to minimize bias.
  • Clean datasets by checking for inconsistencies, errors, and formatting issues in order to prepare them for analysis.
  • Prepare documentation on the data collection and cleaning processes, including any relevant information to contextualize the data to facilitate its reuse.

Data Analysis

Exploring the data, uncover hidden trends and relationship, and apply reliable econometrics techniques for causal inference.

  • Performing Exploratory Data Analysis (EDA).
  • Variables manipulation for detailed analysis.
  • Building statistical models tailored to each business/research question.
  • Assessing modelsâ€™ performance and effectiveness through diagnostics.
  • Creation of clean and reproducible reports using R Notebooks.

Data Visualization

Dataviz is about storytelling and conveying the results of the analysis in an engaging way.

  • Designing infographics with R, Tableau, and Adobe Creative Cloud.
  • Developing web applications with RShiny
  • Creating static and animated graphs for business and scientific publications.
  • Building interactive plots and dashboards.
  • Modelling 2D and 3D spatial models for advanced political and econometric analysis.
